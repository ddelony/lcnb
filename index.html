lcnb by ddelony

Counts lines in standard input or file, omits blank lines.

lcnb takes a filename as an argument or uses standard input, and prints the number of lines, minus the number of blank lines.

Dependencies

Perl 5.8.8

Install

Copy the file into your /bin directory or any other directory accessible in your $PATH
